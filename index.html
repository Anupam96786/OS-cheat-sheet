grep Commands

Consider the file grepText.txt containing the following lines: 

I saw a lame man
in a lane
close to my door.

  • Displays the line containing the pattern lan

    grep "lan" grepText.txt 
    in a lane

  • Displays the line containing the pattern LAne ignoring case

    grep -i “LAne” grepText.txt 
    in a lane

  • Displays number of lines containing the pattern a

    grep -c “a” grepText.txt 
    2

  • Matches whole word only

    grep -w “clo” grepText.txt

    displays nothing

  • matches whole word only

    grep -w “close” grepText.txt 
    close to my door.

  • displays only the word

    grep -o “close” grepText.txt 
    close

  • displays matched line along with line number

    grep -n “close” grepText.txt 
    3:close to my door.

  • displays unmatched limes

    grep -v “close” grepText.txt 
    I saw a lame man
in a lane

  • displays matched line that begins with the given pattern

    grep "^close" grepText.txt 
    close to my door.

  • displays matched line that ends with the given pattern

    grep "man$" grepText.txt 
    I saw a lame man

  • matching multiple patterns

    grep -e "man" -e "close" grepText.txt 
    I saw a lame man
close to my door.

  • Orphan process: An orphan process is a computer process whose parent process has finished or terminated, though it remains running itself.
  • Zombie process: A zombie process is a process whose execution is completed but it still has an entry in the process table. Zombie processes usually occur for child processes, as the parent process still needs to read its child's exit status. Once this is done using the wait system call, the zombie process is eliminated from the process table. This is known as reaping the zombie process.

  • SIGINT is the signal interrupt produced when ctrl + C is pressed.
  • SIGSTOP is the stop signal which causes a process to suspend its execution. This signal is produced when ctrl + Z is pressed.

Using pthread_create function. 

#include <stdio.h>
#include <stdlib.h>
#include <unistd.h>       //Header file for sleep(). man 3 sleep for details.
#include <pthread.h>
    
// A normal C function that is executed as a thread 
// when its name is specified in pthread_create()
void *myThreadFun(void *vargp)
{
    sleep(1);
    printf("Printing GeeksQuiz from Thread \n");
    return NULL;
}
    
int main()
{
    pthread_t thread_id;
    printf("Before Thread\n");
    pthread_create(&thread_id, NULL, myThreadFun, NULL);
    pthread_join(thread_id, NULL);
    printf("After Thread\n");
    exit(0);
}

A program creates threads with the pthread_create function, it waits for them to terminate with the pthread_join function. pthread_join takes only two arguments: thread id to specify the waited thread and pointer to void* where exit status of the specified thread.If the user does not want to retrieve the exit code of the waited thread, NULL value should be passed as the second argument.
